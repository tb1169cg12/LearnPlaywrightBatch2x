# VS Code Keyboard Shortcuts - macOS

## General

| Shortcut | Action |
|----------|--------|
| `Cmd + Shift + P` | Command Palette |
| `Cmd + P` | Quick Open (Go to File) |
| `Cmd + Shift + N` | New Window |
| `Cmd + W` | Close Tab |
| `Cmd + K, Cmd + W` | Close All Tabs |
| `Cmd + \` | Split Editor |
| `Cmd + 1` / `2` / `3` | Focus Editor Group 1 / 2 / 3 |
| `Cmd + Shift + [` | Previous Editor |
| `Cmd + Shift + ]` | Next Editor |
| `Cmd + Option + ←` | Previous Tab |
| `Cmd + Option + →` | Next Tab |
