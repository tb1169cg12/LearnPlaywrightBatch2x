# Learn Playwright Batch 2x

A learning repository for JavaScript fundamentals and Playwright automation testing.

## Repository Structure

```
LearnPlaywrightBatch2x/
├── Chapter_01_Basics/                    # JavaScript basics
│   ├── 01_Basics.js
│   ├── 02_JS.js
│   ├── 03_JS_Command.js
│   └── 04_JS_Hot_code.js
│
├── Chapter_02_Javascript_Concepts/       # Core JS concepts
│   ├── 05_JS_Basics.js
│   ├── 06_JS_identifier rules
│   ├── 07_JS_Keywords_Examples.js
│   ├── 08_JS_Identifier_Rules_Examples.js
│   └── 09_JS_identifier_part2            # Naming conventions (camelCase, snake_case, etc.)
│
└── Chapter_03_Identifier_Literals/       # Tools & shortcuts
    ├── VS_Cdoe_mac_keyboardshortcut.md
    └── VS_Code_windows_keyboardshortcut.md
```

## Topics Covered

- JavaScript Basics (`console.log`, variables)
- Identifiers & Naming Rules
- Reserved Keywords with Examples
- Naming Conventions (camelCase, PascalCase, snake_case, SCREAMING_SNAKE_CASE, kebab-case)
- VS Code Keyboard Shortcuts (macOS & Windows/Linux)

## How to Run

Open any `.js` file in VS Code and run with Node.js:

```bash
node Chapter_01_Basics/01_Basics.js
```

Or use VS Code Run button / terminal.
